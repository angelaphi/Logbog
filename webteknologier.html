Hvad er internettet?

  • Det er en masse individuelle locale servicer der snakker sammen.

  • Min viden kan ære til gavn for andre.
  • Decentraliseret ??

Hvordan flytter internettet information?
  • Binary code (binær-kode) (1,0)
  • Bandwith maximum capacity of a device.
  • Bit rate number of bits that we can send over a given period of time.
  • Latency time it takes for a bit to travel from one place to another.
  • Electricity - Pro: Cheap - Con: Signal loss
  • Light - Pro: Really fast, no signal loss - Con: Expensive, hard to work with
  • Radio - Pro: Totally mobile - Con: Short range
  • Transmisssionmiddel
  • Kobber: billig, kan ikke sende særlig langt
  • Lysleder: hurtigt, kan sende i vinkler
  • Lokal trådløs
  • Global trådløs: satelitter
